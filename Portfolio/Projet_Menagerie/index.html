Projet Java – Ménagerie

Cette application Java avec interface graphique permet de gérer une ménagerie : ajouter, rechercher et supprimer des animaux.

📂 Fichiers Java

  • AjoutAnimalFenetre.java
  • Animal.java
  • Menagerie.java
  • MenuFenetre.java
  • Programme.java
  • RechercheAnimalFenetre.java
  • SuppressionAnimalFenetre.java

⚙️ Fonctionnalités

  • Ajouter un animal à la ménagerie
  • Rechercher un animal par critères
  • Supprimer un animal
  • Interface graphique simple en Java Swing
